No sistema, a gerência dos editais para o sorteio de vagas é feito pela entidade mantenedora. Esses dados são cadastrados pela mantenedora e ficam disponíveis no sistema conforme o que foi estabelecido no edital. Para que possamos atender essa solicitação e inserir o edital com as vagas para cada unidade de ensino, precisamos do envio do edital para nosso e-mail de atendimento (atendimentogr8@gmail.com) e a partir dos dados efetuar o cadastro das vagas para sorteio.
